Скрипт читающий файлы с данными о рейтингах товаров и формирующий отчеты. Формируется один отчёт average-rating (см. пример ниже). Отчёт включает в себя список брендов и средний рейтинг бренда, бренды сортируются по рейтингу. Название файлов (может быть несколько) и название отчета передается в виде параметров --files и --report. Отчёт формируется по всем переданных файлам, а не по каждому отдельно.
